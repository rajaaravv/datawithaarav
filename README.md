# 🚀 About DataWithAarav

**DataWithAarav** is an open-source platform built for anyone who wants to become a **Data Analyst**.

Instead of wasting time searching through hundreds of random YouTube videos and articles, follow one structured roadmap that takes you from **beginner → job-ready**.

Whether you're a student, working professional, or career switcher, DataWithAarav provides everything you need in one place.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/rajaaravv/datawithaarav.git
```

## Navigate

```bash
cd datawithaarav
```

## Install Dependencies

```bash
npm install
```

or

```bash
bun install
```

## Run Development Server

```bash
npm run dev
```

or

```bash
bun dev
```

Open

```
http://localhost:3000
```
